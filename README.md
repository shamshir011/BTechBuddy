# 📚 BTech Buddy App

BTech Buddy is an Android application designed to help B.Tech students with academic preparation and career guidance. The app provides programming quizzes, important questions, and structured roadmaps for different development domains.

---

## ✨Features

- **Programming Quizzes**  
  Practice quizzes for Java and Python with important questions and answers.

- **Important Questions**  
  Curated questions to help students prepare for exams and interviews.

- **Learning Roadmaps**  
  Step-by-step roadmaps for:
  - Full Stack Web Development  
  - Android Development  

- **User-Friendly Interface**  
  Clean and simple UI using modern Android components.

- **Firebase Integration**  
  Real-time database used to store and fetch quiz and learning content.

- **Monetization**  
  Integrated banner ads.

---

## Tech Stack

- **Language:** Kotlin  
- **Backend:** Firebase Realtime Database  
- **UI:** XML, RecyclerView, CardView, Material Design  
- **IDE:** Android Studio  

---

## Purpose

This project was built to support B.Tech students by providing a single platform for learning, practicing, and exploring career paths in tech.
